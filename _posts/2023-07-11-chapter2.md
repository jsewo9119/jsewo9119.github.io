#  C++ 프로그램의 구성요소

'''C++

//myfirst.cpp

#include <iostream>

int main()

{

​	using namespace std;

​	cout << "C++의 세계로 오십시오.";

​	cout << endl;

​	cout << "후회하지 않으실 겁니다!" << endl;

​	return 0;

}

'''

- //로 시작되는 주석문
- #include 전처리 지시자
- 함수머리:int main()
- Using namespace 지시자
- {와}로 범위가 정해지는 함수 몸체
- C++의 cout 기능을 사용하여 메지시를 출력하는 구문들
- main() 함수를 종료하는 return 기능
